# Cascading Navigation Bar

A responsive, neumorphic **Cascading Navigation Bar** built for the EaseMotion CSS component library.

## Features

- Pure HTML & CSS
- Responsive Layout
- Accessibility Support
- Keyboard Navigation
- Cascading Entrance Animation
- Neumorphic Design
- Uses EaseMotion CSS variables
- Uses custom EaseMotion keyframes
- No JavaScript Required
